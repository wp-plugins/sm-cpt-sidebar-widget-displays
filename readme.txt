== Description ==

Enables a widget that can display multiple posts of any post type or CPT (custom post type) into any sidebar. It uses html tag classes for easy styling, has some sorting options, & has some advanced "taxonomy matching filters", perfect use case is for brief testimonials. This plugin is maintained and provided by http://sethmatics.com/.

We plan to expand the feature set of this plugin to work with more post types in the future. Some ideas are uses with featured images, link options, etc. Please use the support forum to bring your suggestions so that we can create some feature milestones for the dev team.

== Installation ==

To install the plugin manually:

1. Extract the contents of the archive (zip file)
2. Upload the sm-cpt-sidebar-widget-displays folder to your '/wp-content/plugins' folder
3. Activate the plugin through the Plugins section in your WordPress admin
4. Visit the "appearance->widgets" section from within /wp-admin/.
5. Drag the "SM Display CPTs" widget to a sidebar and configure the options. 
**repeat into as many sidebars or multiple times in 1 sidebar for multiple post types
6. View a page that uses the above sidebar to view the results. Most likely you will need to enhance the widget using styles in the themes style.css.
